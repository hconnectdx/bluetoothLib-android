# bluetoothLib-android

## 소개

`bluetoothLib-android`는 블루투스 기술을 안드로이드 애플리케이션에 원활하게 통합하는 데 필요한 모든 도구와 기능을 제공합니다.
이 라이브러리는 기기 검색, 페어링, 연결 및 블루투스 지원 기기 간의 데이터 교환 프로세스를 간소화합니다.
